## üìñ Overview

**AI News Trader** is a sophisticated Telegram bot that bridges the gap between raw chart data and AI-driven news analysis. Unlike standard scrapers, it uses a **hybrid approach**:

1.  **Technical Engine (Python):** Calculates exact support levels, detects "3-3-3" reversal patterns, and monitors 52-week high drops using `yfinance` and `scipy`.
2.  **Fundamental Engine (Gemini 2.5 Pro):** Parses specific URL contexts (News/Earnings reports) to explain *why* a move is happening and validates technical targets against analyst consensus.

---

## üöÄ Key Features

### 1. üß† Intelligent "3-3-3" Reversal Detection
Instead of random buying, the bot looks for a specific high-probability setup:
* **Trend:** At least 3 consecutive days of lower lows (The Fall).
* **Signal:** A "Reversal Candle" where Close > Yesterday's High.
* **Confirmation:** Volume analysis to confirm institutional interest.

### 2. üìâ "Best Buy" Limit Calculator
* Uses `scipy.signal.argrelextrema` to find historical support levels (valleys).
* Automatically suggests a **Limit Order price** below the CMP (Current Market Price) to catch wicks.

### 3. üì∞ Gemini 2.5 Pro Integration
* **URL Context:** You pass a link (MoneyControl, TradingView, Google Finance), and the AI "reads" the page to extract news.
* **FUT Target:** Generates a 30-day "Future Target" based on the merged Technical + Fundamental view.

‚òÅÔ∏è Deployment (Cloudflare + VPS)
Since scipy and pandas are too heavy for serverless functions (like Cloudflare Workers), this project uses a Proxy Architecture:

Cloudflare Worker (worker.js): Receives the Telegram Webhook -> Protects IP -> Forwards to VPS.

VPS (Render/Heroku/DigitalOcean): Runs the heavy Python code (main.py).

Deploying the Cloudflare Worker
Copy the code from worker.js.

Create a new Worker in Cloudflare Dashboard.

Add Environment Variables: PYTHON_BACKEND_URL (Your VPS URL) and SECRET_TOKEN.
